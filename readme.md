## 🏠 Atividade de Casa

### 🟣 1. Projeto “Meus Filmes Favoritos” 🎬

**Passos sugeridos:**

1. Criar um `README.md` com:
   - Um **título**
   - Uma **lista com pelo menos 5 filmes**
   - Um **link para o trailer** de um deles
   - Uma **imagem do seu favorito**
